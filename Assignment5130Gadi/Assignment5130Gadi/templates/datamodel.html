My Data Model

In my project, I have used 3 different data sets, each including every game of either Michael Jordan's, LeBron James' or Kobe Bryant's careers. In each data set you can find data on details regarding the game played (date, location etc.) and statistics of the player from that game (Point, Rebounds, Assists and more).

Jordan Dataset
Michael Jordan's career game log, includes 1251 games played (1072 regualer season, 179 playoffs).
LeBron Dataset
LeBron James' career game log, includes 1437 games played (1198 regular season, 239 playoffs).
Kobe Dataset
Kobe Bryant's career game log, includes 1566 games played (1346 regular season, 220 playoffs
